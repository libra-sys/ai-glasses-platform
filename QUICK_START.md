## 👤 普通用户快速开始

### 第一步：注册账号
1. 访问平台首页
2. 点击右上角"登录/注册"
3. 切换到"注册"标签
4. 填写用户名和密码
5. 点击"注册"按钮
6. ✅ 无需邮箱验证，立即可用

### 第二步：浏览组件
1. 点击导航栏"组件库"
2. 浏览所有已发布的组件
3. 使用搜索框查找特定组件
4. 点击组件卡片查看详情

### 第三步：下载组件
1. 进入组件详情页
2. 点击"下载组件"按钮
3. 组件文件自动下载到本地

### 第四步：发布组件
1. 点击右上角用户菜单
2. 选择"发布组件"
3. 填写组件信息：
   - 组件名称
   - 组件描述
   - 选择分类
   - 上传组件文件
   - 上传封面图片（可选）
4. 点击"发布组件"
5. 等待管理员审核

### 第五步：使用AI工具
1. 点击导航栏"AI工具"
2. 选择需要的工具：
   - **AI对话**：获取开发建议
   - **AI搜索**：搜索技术资料
   - **网页总结**：总结网页内容
   - **图片生成**：生成组件封面

## 👨‍💼 管理员快速开始

### 前提条件
- 已在数据库中设置为管理员（role='admin'）
- 详见 `ADMIN_SETUP.md`

### 访问管理后台
1. 登录管理员账号
2. 点击右上角用户头像
3. 选择"管理后台"
4. 进入管理界面

### 审核组件
1. 在"组件管理"标签中
2. 查看待审核的组件
3. 点击"通过"或"拒绝"
4. 组件状态立即更新

### 发布公告
1. 切换到"公告管理"标签
2. 点击"发布新公告"
3. 填写标题和内容
4. 选择优先级
5. 点击"发布公告"

### 管理用户
1. 切换到"用户管理"标签
2. 查看所有注册用户
3. 查看用户角色和信息

## 🔧 常用操作

### 搜索组件
```
1. 进入组件库页面
2. 在搜索框输入关键词
3. 按回车或点击搜索按钮
4. 查看搜索结果
```

### 收藏组件
```
1. 进入组件详情页
2. 点击"收藏"按钮
3. 在个人中心查看收藏列表
```

### 评价组件
```
1. 进入组件详情页
2. 滚动到评论区
3. 选择评分（1-5星）
4. 填写评论内容
5. 点击"提交评论"
```

### 管理我的组件
```
1. 进入个人中心
2. 切换到"我的组件"标签
3. 查看所有发布的组件
4. 查看审核状态
```

### 使用AI搜索
```
1. 进入AI工具页面
2. 切换到"AI搜索"标签
3. 输入搜索关键词
4. 点击"搜索"
5. 查看AI总结的结果
```

### 生成组件封面
```
1. 进入AI工具页面
2. 切换到"图片生成"标签
3. 输入详细的图片描述
   例如："一个科技感的AI眼镜图标，蓝色主题，简约风格"
4. 点击"生成图片"
5. 等待30-60秒
6. 右键保存生成的图片
```

## 💡 使用技巧

### 组件发布技巧
1. **清晰的命名**：使用简洁明了的组件名称
2. **详细的描述**：说明组件功能、使用方法、注意事项
3. **选择正确的分类**：便于用户查找
4. **上传封面图**：使用AI工具生成专业的封面图
5. **测试后发布**：确保组件文件完整可用

### AI工具使用技巧
1. **AI对话**：
   - 提问要具体明确
   - 可以多轮对话深入讨论
   
2. **AI搜索**：
   - 使用具体的技术关键词
   - 等待流式输出完成
   
3. **网页总结**：
   - 确保URL完整（包含http://或https://）
   - 选择公开可访问的网页
   
4. **图片生成**：
   - 描述越详细效果越好
   - 包含风格、颜色、主题等信息
   - 耐心等待30-60秒

### 管理员操作技巧
1. **审核组件**：
   - 优先审核等待时间长的组件
   - 检查组件名称和描述是否规范
   - 对于不合格的组件使用"拒绝"而不是"删除"

2. **发布公告**：
   - 重要公告使用"高"优先级
   - 标题简洁（20字以内）
   - 内容清晰完整
   - 定期清理过期公告

3. **用户管理**：
   - 定期查看新注册用户
   - 关注异常账号
   - 谨慎设置管理员权限

## ❓ 常见问题

### Q: 注册后需要验证邮箱吗？
A: 不需要，注册后可以立即登录使用。

### Q: 组件审核需要多久？
A: 取决于管理员的审核速度，通常在24小时内完成。

### Q: 可以修改已发布的组件吗？
A: 目前不支持修改，如需更新请重新发布新版本。

### Q: 图片生成失败怎么办？
A: 请检查网络连接，确保描述详细，等待足够时间（30-60秒），然后重试。

### Q: 如何成为管理员？
A: 管理员需要在数据库中手动设置，详见 `ADMIN_SETUP.md`。

### Q: 忘记密码怎么办？
A: 目前不支持密码找回，请联系管理员重置密码。
