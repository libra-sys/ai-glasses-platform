# 一键设置管理员账号

## 🚀 快速设置（3步完成）

### 第1步：注册4个账号

访问平台，依次注册以下账号：

| 用户名 | 密码 |
|--------|------|
| user1  | 123456 |
| user2  | 123456 |
| user3  | 123456 |
| user4  | 123456 |

**注意**：每注册一个账号后，需要退出登录，再注册下一个。

---

### 第2步：执行SQL设置管理员

打开Supabase控制台 → SQL Editor → 复制粘贴以下代码 → 点击Run：

```sql
UPDATE profiles 
SET role = 'admin'::user_role 
WHERE username IN ('user1', 'user2', 'user3', 'user4');
```

---

### 第3步：验证

使用 `user1` / `123456` 登录，应该能看到"管理后台"菜单。

---

## ✅ 完成！

现在可以使用任一管理员账号登录并管理平台了。

---

## ❓ 遇到问题？

### 问题1：提示"Invalid login credentials"
**原因**：账号还没注册

**解决**：先完成第1步，注册账号

---

### 问题2：登录成功但看不到"管理后台"
**原因**：没有执行第2步的SQL

**解决**：在Supabase控制台执行SQL设置管理员权限

---

### 问题3：不知道如何访问Supabase控制台
**步骤**：
1. 访问 https://supabase.com
2. 登录你的账号
3. 选择对应的项目
4. 点击左侧菜单的"SQL Editor"

---

### 问题4：SQL执行后仍然不是管理员
**验证**：执行以下SQL检查：
```sql
SELECT username, role FROM profiles 
WHERE username IN ('user1', 'user2', 'user3', 'user4');
```

应该看到4条记录，role都是'admin'。

如果不是，重新执行第2步的SQL。
